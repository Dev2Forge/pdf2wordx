Este proyecto usa Tkinter V8.6 como modelo de interfaz gráfica, pdf2docx V0.5.8 para realizar las conversiones de PDF a DOCX (Leer pdf2docx library). Si alguien conoce una librería más estable y mejor que 'pdf2docx' podría decírmelo para mejorar la calidad de este programa.

    Características:

  • Convertir Documento PDF A WORD
  • Interfaz Gráfica (No debe lidiar con una terminal)
  • Fácil de usar
  • Código Abierto
  • No roba la información
  • Fuera de línea
  • Trabaja Con Python Localmente
  • No Sube los documentos a la nuve, laconversión es local

PDFTOWORD es un programa que permite realizar la conversión de documentos PDF con contenidos de imágenes y texto a un formato .docx editable. Este programa es una solución efectiva para aquellos que no queremos sufrir PÉRDIDA de información y no queremos subir nuestros PDF a los servicios online, ya que no sabemos a ciencia cierta a donde viaja nuestra información y que pasa con esta, no sabemos si se hacen copias o quedan "rondando" por internet a disposición de desconocidos.
PDFTOWORD realiza todo su trabajo fuera de línea, no se requiere acceso a internet, tampoco es necesario registrarse (ni más faltaba), usé la librería pdf2docx por su simplicidad a la hora de convertir un archivo, además que su conversión es muy buena ya que el diseño del PDF será 99.9% el diseño del docx, claro está que pueden suceder errores, tales como Error En La Conversión debido en algunos casos a el formato de imagen con el cual fue creado el PDF.
Los errores "Conocidos" se listan en esta sección al igual que los "límites" de conversión (si hay "posibles límites" necesarios para evitar el error), de igual manera y por estúpido que parezca, sí, hay una sección de "¿Cómo usar el programa?".

    Límites

  • Archvios grandes (más de 20MB) podrían tener complicaciones, la conversión podría terminar exitosamente ó posiblemente con errores
  • Los formatos NO RGB (formato de color de Imagen) darán un Error de conversión
  • Sólo se permite la conversión de un archivo a la vez

Usar PDF A WORD es muy fácil. Antes que nada se debe descargar el programa (Ya sea código fuente o exe - Mayor tamaño), según la opción descargada (código - exe) siga los pasos:

  • Si descargó el código fuente:
    1. Debe tener python instalado y la librería pdf2docx
      #Con python instalado, ejecute CMD (Win+R, cmd) 
      pip install pdf2docx==0.5.8
    2. Una vez instalado, descomprimir el zip descargado, ejecutar el script "main.py" (ya sea desde la terminal o doble click directo)
  • Si descargó el exe:
    1. Ejecute el archivo pdftoword.exe
  • Para ambos casos:
    1. Una vez la interfaz gráfica cargue, lo primero es darle un nombre al archivo que será guardado con la extensión .docx
    2. Seleccionar el archivo danco click en el botón Abrir Archivo
    3. Seleccionar directorio/carpeta en la cual se guardará el archivo .docx con el botón Elegir Directorio
    4. Finalmente, de click en el botón Convertir

  • Error en conversión por imagen no RGB (Sólo se admiten imágenes con formatos RGB dentro del pdf)
  • El programa entra en modo "EL programa no responde" (en windows) ya que la interfaz se congela durante la conversión del archivo
